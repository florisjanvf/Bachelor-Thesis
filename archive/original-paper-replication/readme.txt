This folder contains all the code and results for the replication of the paper by Peeters and Van Ours (2021) which I am
extending in my bachelor thesis. The "repository-peeters-van-ours-2021" directory is empty except for the .gitignore
file. If the user wants to run the "replication_peeters_van_ours_2021" notebook they will need to request access to the
original repository and copy the files into the "repository-peeters-van-ours-2021" folder. I have not included these
files in this repository to respect intellectual property rights.
